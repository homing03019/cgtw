# 調色盤格式 (CGP)

x-gate 使用 8 位元索引顏色的調色盤（Indexed Color Palette），總共包含 256 種顏色。

## 調色盤構成

調色盤的 256 個顏色是由 **固定色** 與 **自定義色** 組成的：

1. **Prefix (前 16 色)**:
   - 索引 0-15。
   - 固定預定義的 16 個顏色。
   - 索引 0 (RGB 0,0,0) 為透明色。
2. **Custom (中間 224 色)**:
   - 索引 16-239。
   - 儲存在 `.cgp` 檔案中。
   - 每個顏色佔用 3 個位元組 (BGR 順序)。
   - 檔案大小固定為 `(256 - 32) * 3 = 672` 位元組。
3. **Suffix (後 16 色)**:
   - 索引 240-255。
   - 固定預定義的 16 個顏色。

## 實作細節

在 `pkg/palette.go` 中，`NewPaletteFromCGP` 函數會讀取 672 位元組的自定義顏色，並將其與預定義的 `prefix` 和 `suffix` 數組拼接，形成一個完整的 256 色調色盤。

### 透明處理
- 若顏色值為 RGB(0, 0, 0)，程式會將其 Alpha 通道設為 0，視為透明色。
- 其他顏色 Alpha 通道一律設為 0xff (255)。
- 部分特定顏色（如純紅 0xff, 0, 0、純綠 0, 0xff, 0、純藍 0, 0, 0xff）也會被視為透明色處理。
