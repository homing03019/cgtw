# 契约召唤宠 — 可学技能清单

规则（拟采用）：
- **可学**：skill.txt 学习栏 `0`（宠物/抗性）、`2`（战斗共用）、`4`（补血/辅助）
- **不可学**：`3`（玩家专用，如元素祈祷）、`field≥5`（Boss技）
- 弓宠额外固定：**攻击(73)**；池内 **射击(73)**
- 五围流派会决定从哪些**技能池**抽取；每场随机 0–3 招 + 可能强制 AOE
